# PRINSIP TEMPLATE

Template ini bukan formulir administrasi.

Template ini adalah alat validasi yang memastikan:

* seluruh perubahan dapat ditelusuri;
* tidak ada perubahan di luar mandat editor;
* TP revisi siap masuk Audit Ulang.

Ukuran keberhasilan bukan banyaknya perubahan yang dilakukan,

melainkan kemampuan editor menghasilkan TP revisi yang sepenuhnya sesuai Output Audit dan siap diproses menjadi **TP Final** tanpa penyesuaian tambahan.
