# DOKUMEN 14 — WORKFLOW CLAUDE CODE MENUJU TP FINAL
Versi 3.0

## TUJUAN
Workflow ini menetapkan alur kerja standar Claude Code
untuk menghasilkan TP final yang siap live di browser.

## FILE
docs/output-v5/tp-xx-v5.js
(diedit langsung menggunakan str_replace)

Claude Code tidak membuat file baru.
Claude Code tidak membuat folder baru.

## ALUR KERJA

Langkah 1 — Baca
Baca tp-xx-v5.js secara utuh.

Langkah 2 — Audit
Lakukan audit sesuai Audit OS.
Simpan seluruh temuan dalam memori.

Langkah 3 — Edit
Terapkan seluruh temuan yang dapat diterapkan.
Simpan seluruh perubahan dalam memori.

Langkah 4 — Quality Gate
Jalankan seluruh Quality Gate Audit OS dan Editor OS
dalam memori.

Langkah 5 — Keputusan

Jika seluruh temuan dapat diterapkan dan
seluruh Quality Gate lulus:
→ langsung ke Langkah 6.

Jika ada temuan yang tidak dapat diterapkan:
→ tampilkan temuan tersebut;
→ berikan rekomendasi perbaikan;
→ tunggu keputusan owner + Claude (chat).

  Jika TULIS  → lanjut ke Langkah 6.
  Jika PERBAIKI → revisi → tampilkan lagi → tunggu.

Langkah 6 — Simpan
Terapkan seluruh perubahan pada tp-xx-v5.js
menggunakan str_replace.
Jangan membuat file baru.

Langkah 7 — Service Worker
Lakukan bump SW / cache version
sesuai mekanisme proyek.

Langkah 8 — Git
1. Commit dengan pesan yang sesuai.
2. Push ke remote repository.
3. Pastikan push berhasil.

## HASIL AKHIR
tp-xx-v5.js yang telah direvisi langsung,
siap live di browser.
