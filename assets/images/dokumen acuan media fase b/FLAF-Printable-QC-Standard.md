## A. Standar Umum (berlaku untuk SEMUA jenis media)

### A1. Teknis File

- [ ] **[WAJIB]** Format: PNG
- [ ] **[WAJIB]** Nama file sesuai konvensi: `tp-bNN-nama-kebab.png` — huruf kecil, tanpa spasi
- [ ] **[WAJIB]** Nama file cocok persis dengan field `file` di daftar final v2
- [ ] **[WAJIB]** Background: putih bersih (#FFFFFF)
- [ ] **[WAJIB]** Tidak ada watermark, logo, atau teks "AI Generated"

**Spesifikasi dimensi (diverifikasi dari Fase A):**

| Jenis | Dimensi | Rasio | Catatan |
|---|---|---|---|
| FC | **1254 × 1254 px** | 1:1 | Semua FC termasuk sequence-cards B19 |
| WS, REF, POST, TEXT | **1054 × 1492 px** | ≈ A4 | — |

> **Pengecualian B19 sequence-cards:** meski diklasifikasi FC, kanvasnya **1054 × 1492 px** (A4)
> karena berisi 6 panel dalam 1 lembar. Ini satu-satunya FC yang tidak square.

- [ ] **[WAJIB]** Dimensi sesuai tabel di atas — toleransi ±10 px
- [ ] **[WAJIB]** Mode warna: **PNG-8 (8bpp Indexed)** — konsisten dengan Fase A
- [ ] **[DIANJURKAN]** Ukuran file di bawah 700 KB

### A2. Gaya Visual

Fase A menggunakan gaya **semi-realistis + karakter kartun/anime** — wajib direplikasi di Fase B.

- [ ] **[WAJIB]** Background: painterly semi-realistis, detail kaya
- [ ] **[WAJIB]** Karakter: kartun/anime — proporsi sedikit chibi, ekspresi ekspresif dan ramah
- [ ] **[WAJIB]** Setting Indonesia: rumah atap genteng merah, lingkungan kampung/kota lokal, vegetasi tropis
- [ ] **[WAJIB]** Karakter anak SD: rambut hitam lurus, kulit sawo matang, mata besar
- [ ] **[WAJIB]** Seragam sekolah: baju putih + celana/rok merah, sepatu hitam, tas ransel
- [ ] **[WAJIB]** Pakaian rumah: kasual lokal (kaos, batik, daster) — bukan pakaian Barat generik
- [ ] **[WAJIB]** Adegan kontekstual penuh — bukan objek terisolasi di background polos
- [ ] **[WAJIB]** Pencahayaan sesuai konteks waktu:
  - Morning: golden hour, langit biru cerah
  - Afternoon: terik dari atas, bayangan pendek
  - Evening: sunset oranye-kemerahan
  - Night: langit gelap, bulan sabit + bintang, lampu rumah menyala
- [ ] **[WAJIB]** Warna vivid dan saturasi tinggi — natural, bukan neon
- [ ] **[WAJIB]** Terbaca jelas saat dicetak hitam-putih
- [ ] **[WAJIB]** Bebas dari stereotip tidak pantas
- [ ] **[DIANJURKAN]** Konsistensi karakter antar TP

### A3. Bahasa & Teks

- [ ] **[WAJIB]** Ejaan bahasa Inggris benar — tidak ada typo
- [ ] **[WAJIB]** Font terbaca pada ukuran cetak
- [ ] **[WAJIB]** Tidak ada teks bahasa Indonesia di konten utama
  - Pengecualian: instruksi WS yang memang bilingual
- [ ] **[DIANJURKAN]** Font sans-serif bersih untuk teks utama

### I1. Spesifikasi teknis
```
Dimensi: 1254 × 1254 px (flash card)
      ATAU 1054 × 1492 px (worksheet / REF / poster / teks cetak / sequence cards)
Format: PNG, background putih bersih #FFFFFF
```

### I2. Blok gaya visual (copy-paste di setiap prompt)
```
GAYA WAJIB — konsisten dengan seri FLAF Fase A:

Background: painterly semi-realistis, detail kaya
Karakter: kartun/anime — proporsi chibi, ekspresi ekspresif dan ramah
Setting: Indonesia — rumah atap genteng merah, dinding putih/krem, teras
  berkeramik, jalanan kampung, pohon tropis (pisang, kelapa, bougainvillea)
Interior: ruang keluarga sederhana, lemari kayu, karpet motif, lampu gantung

Karakter anak SD: rambut hitam lurus, kulit sawo matang, mata besar
Seragam sekolah: baju putih + celana/rok merah, sabuk hitam, sepatu hitam, ransel
Pakaian rumah: kasual lokal (kaos, batik, daster) — BUKAN pakaian Barat

Pencahayaan:
  Morning  : golden hour kanan bawah, langit biru cerah
  Afternoon: terik dari atas, bayangan pendek
  Evening  : sunset oranye-merah, langit gradasi
  Night    : langit gelap, bulan sabit + bintang, lampu rumah kuning hangat

Warna: vivid, saturasi tinggi, natural — bukan neon
Tidak ada watermark, teks AI Generated, atau logo apapun
```
