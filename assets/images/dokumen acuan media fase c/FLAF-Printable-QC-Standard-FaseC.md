## A. Standar Umum (berlaku untuk SEMUA jenis media)

### A1. Teknis File

- [ ] **[WAJIB]** Format: PNG
- [ ] **[WAJIB]** Nama file sesuai konvensi: `tp-c{NN}-nama-kebab.png` — huruf kecil, tanpa spasi
- [ ] **[WAJIB]** Nama file cocok persis dengan field `file` di daftar final v1
- [ ] **[WAJIB]** Background: putih bersih (#FFFFFF)
- [ ] **[WAJIB]** Tidak ada watermark, logo, atau teks "AI Generated"

**Spesifikasi dimensi:**

| Jenis | Dimensi | Rasio | Catatan |
|---|---|---|---|
| FC | **1254 × 1254 px** | 1:1 | Flash card kosakata |
| SORT | **1254 × 1254 px** | 1:1 | Sorting/game card — dipotong |
| WS, REF, TEXT, POST | **1054 × 1492 px** | ≈ A4 | — |

- [ ] **[WAJIB]** Dimensi sesuai tabel di atas — toleransi ±10 px
- [ ] **[WAJIB]** Mode warna: **PNG-8 (8bpp Indexed)**
- [ ] **[DIANJURKAN]** Ukuran file di bawah 700 KB

### A2. Gaya Visual

Fase C menggunakan gaya visual **identik** dengan Fase A & B.
Karakter kelas 5–6 SD: sedikit lebih tinggi dari kelas 3–4, tapi
tetap proporsi chibi/kartun — bukan remaja.

- [ ] **[WAJIB]** Background: painterly semi-realistis, detail kaya
- [ ] **[WAJIB]** Karakter: kartun/anime — proporsi chibi, ekspresi ekspresif
- [ ] **[WAJIB]** Setting Indonesia: rumah atap genteng merah, lingkungan lokal, vegetasi tropis
- [ ] **[WAJIB]** Karakter anak SD: rambut hitam lurus, kulit sawo matang, mata besar
- [ ] **[WAJIB]** Seragam sekolah: baju putih + celana/rok merah, sepatu hitam, tas ransel
- [ ] **[WAJIB]** Pakaian rumah: kasual lokal (kaos, batik, daster) — bukan pakaian Barat
- [ ] **[WAJIB]** Adegan kontekstual penuh — bukan objek terisolasi di background polos
- [ ] **[WAJIB]** Warna vivid, saturasi tinggi, natural — bukan neon
- [ ] **[WAJIB]** Terbaca jelas saat dicetak hitam-putih
- [ ] **[WAJIB]** Bebas dari stereotip tidak pantas
- [ ] **[DIANJURKAN]** Konsistensi karakter antar TP

### A3. Bahasa & Teks

- [ ] **[WAJIB]** Ejaan bahasa Inggris benar — tidak ada typo
- [ ] **[WAJIB]** Font terbaca pada ukuran cetak
- [ ] **[WAJIB]** Tidak ada teks bahasa Indonesia di konten utama
  - Pengecualian: instruksi WS yang memang bilingual, dan teks menu (nama makanan Indonesia)
- [ ] **[DIANJURKAN]** Font sans-serif bersih untuk teks utama

## F. Sorting / Game Card (SORT) — 5 file

**Kanvas:** 1254 × 1254 px · PNG-8 · 1:1
*(Sama dengan FC — kartu persegi yang dipotong)*

### Checklist SORT
- [ ] **[WAJIB]** Satu adegan per kartu — mudah diidentifikasi tanpa teks
- [ ] **[WAJIB]** Nomor urut **TIDAK dicetak** — siswa yang mengurutkan
- [ ] **[WAJIB]** Ilustrasi mengisi penuh area
- [ ] **[WAJIB]** Garis potong jelas jika dicetak dalam satu lembar
- [ ] **[DIANJURKAN]** Nomor kecil di sisi belakang untuk self-check

**Catatan khusus C14 — Juice Steps:**
- 5 kartu = 5 langkah membuat jus jeruk
- Step 1: Ambil jeruk → Step 2: Belah dua → Step 3: Peras ke gelas → Step 4: Tambah gula → Step 5: Aduk dan nikmati
- Dicetak 1 set per pasangan → dipotong → dikocok → diurutkan

## G. Proses Review & Masuk Codebase

```
1. Generate di ChatGPT
        ↓
2. Cek mandiri dengan checklist dokumen ini
        ↓
3. Kirim ke reviewer (owner project)
        ↓
4. Lolos semua [WAJIB]?
   YA  → simpan ke assets/images/printables_fase_c/
   TIDAK → revisi → ulangi dari langkah 2
        ↓
5. printables[] sudah diisi di codebase (selesai di sesi Juni 2026)
        ↓
6. Setelah batch per TP lolos → update sw.js cache list
        ↓
7. Test di browser:
   Layar Mengajar → tab Materi → tombol Cetak Kartu Persiapan
```
